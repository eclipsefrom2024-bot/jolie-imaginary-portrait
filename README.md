# My Imaginary Portrait — Jolie Museum

A single-file front-end prototype for a mobile-first interactive museum activity.

## What it does
- Lets visitors create an original geometric portrait from original SVG assets.
- Drag, scale, rotate, layer, reset, randomize, undo.
- Exports the finished composition as PNG.
- Runs with plain HTML, CSS, JavaScript: no backend, no account, no database.

## Run locally
Just double-click `index.html` to open it in a browser.

For best testing on a phone, use a local web server:
- VS Code + Live Server extension, or
- `python -m http.server 8000` in this folder, then visit `http://localhost:8000`

## Deploy with GitHub Pages
1. Create a new GitHub repository.
2. Upload the files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose branch `main` and folder `/ (root)`.
6. Save. GitHub will provide the public site URL after deployment.

## Before public launch
- Replace the logo placeholder / exhibition wording if needed.
- Confirm final event dates and café offer terms.
- Test downloads on iPhone Safari and Android Chrome.
- Use only original assets or assets with documented rights.
